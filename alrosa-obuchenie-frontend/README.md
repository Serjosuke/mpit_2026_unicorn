# Алроса Обучение — frontend

## Запуск
```bash
cp .env.example .env
docker compose up --build
```

Frontend поднимается на `http://localhost:3000`.

Backend ожидается на `http://localhost:8000`.

## Что есть
- login / register
- дашборд сотрудника
- каталог курсов
- карточка курса
- мое обучение
- внешний курс / заявка
- сертификаты
- уведомления
- календарь
- личный кабинет
- панель согласований для manager / hr

## Важно
В текущем backend нет отдельного Outlook API, нет upload файлов и нет полноценного course detail payload с модулями/уроками.
Во frontend это аккуратно учтено:
- календарь собран из курсов и внешних заявок
- загрузка сертификата заменена созданием сертификата через backend endpoint
- детальная страница курса работает по доступным данным API
